# [C-001] 业务规则提供

> 如何向AI清晰地提供业务规则

**效果提升**：业务理解准确率 +60%

---

## 📝 模板

```markdown
## 业务规则

### 规则来源
- 文档：[docs/business/xxx.md]
- 章节：[具体章节]

### 核心规则
1. **规则名称**：[规则描述]
   - 公式：[如果有]
   - 约束：[限制条件]
   - 示例：[具体例子]

2. **规则名称**：...

### 边界条件
- 当X=0时：[处理方式]
- 当缺少数据时：[处理方式]
- 当异常情况时：[处理方式]

### 业务术语
- 术语1：[解释]
- 术语2：[解释]
```

---

## 💡 关键要素

1. **引用权威文档**：docs/business/中的定义
2. **提供计算公式**：准确的数学表达式
3. **说明边界条件**：null-safe规则
4. **解释业务术语**：避免AI误解
