# [BP-001] 与AI协作流程

> 高效的AI辅助开发工作流程

**效果提升**：整体效率 +40%

---

## 🔄 完整流程

### 1. 任务准备
- [ ] 明确任务目标
- [ ] 收集必要的Context
- [ ] 查询知识库是否有类似案例

### 2. 编写Prompt
- [ ] 使用[P-001] Context提供模式
- [ ] 复杂任务使用[P-002] 任务分解模式
- [ ] 提供参考代码和文档

### 3. 执行任务
- [ ] 提交Prompt给AI
- [ ] 验证AI的理解
- [ ] 如有偏差，立即补充Context

### 4. 验证结果
- [ ] 检查代码是否符合要求
- [ ] 运行测试
- [ ] Code Review

### 5. 记录经验
- [ ] 成功：记录有效的Prompt模式
- [ ] 失败：使用`/ai-evolve record`记录问题
- [ ] 分析：使用`/ai-evolve analyze`提炼模式

---

## 💡 关键原则

1. **迭代优化**：第一次可能不完美，持续改进Prompt
2. **及时反馈**：发现问题立即告知AI
3. **记录积累**：好的坏的都要记录
4. **模式复用**：成功的Prompt要保存到知识库
