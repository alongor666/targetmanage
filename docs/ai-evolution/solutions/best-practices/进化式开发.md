# [BP-003] 进化式开发

> 持续学习、不断优化的开发方式

**效果提升**：长期效率持续提升

---

## 🔄 进化循环

```
遇到问题 → 记录问题 → 分析本质 → 提炼模式 → 应用模式 → 避免重复错误
    ↑                                                              |
    └───────────────────── 进化反馈循环 ──────────────────────────┘
```

---

## 📊 进化指标

### 短期（每周）
- 记录的问题数量
- 解决问题的平均时间
- Prompt一次成功率

### 中期（每月）
- 提炼的模式数量
- 模式应用次数
- 知识库查询命中率

### 长期（每季度）
- 整体开发效率提升
- 代码质量改善
- AI协作满意度

---

## 💡 持续改进建议

1. **每周回顾**：查看本周的问题记录
2. **每月总结**：生成进化报告
3. **每季度优化**：更新Prompt模式库
