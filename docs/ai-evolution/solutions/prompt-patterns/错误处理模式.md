# [P-003] 错误处理模式

> **核心理念**：系统化地提供错误信息，让AI快速定位问题

**效果提升**：问题定位速度 +50%
**适用场景**：Bug修复、错误处理
**难度**：⭐ 简单

---

## 📝 Prompt模板

```markdown
修复以下错误

## 错误信息
```
[完整的错误堆栈]
```

## 错误位置
- 文件：[file_path:line_number]
- 函数：[functionName]

## 复现步骤
1. ...
2. ...

## 期望行为
[应该发生什么]

## 实际行为
[实际发生了什么]

## Context
[相关代码、业务逻辑]

## 已尝试的解决方案
1. 尝试A：[结果]
2. 尝试B：[结果]
```

---

## 💡 关键要素

1. **完整错误堆栈**：不要截断
2. **精确位置**：文件路径 + 行号
3. **可复现步骤**：让AI理解如何触发错误
4. **期望vs实际**：明确差异
