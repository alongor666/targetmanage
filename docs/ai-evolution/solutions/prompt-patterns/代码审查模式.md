# [P-004] 代码审查模式

> **核心理念**：提供审查清单，让AI进行全面的代码检查

**效果提升**：代码质量提升 +30%
**适用场景**：Code Review、质量检查
**难度**：⭐⭐ 中等

---

## 📝 Prompt模板

```markdown
审查以下代码

## 代码
```[language]
[code here]
```

## 审查清单

### 功能性
- [ ] 是否实现了所有需求？
- [ ] 边界条件是否处理？
- [ ] 错误处理是否完善？

### 代码质量
- [ ] 命名是否清晰？
- [ ] 是否遵循项目规范？
- [ ] 是否有重复代码？

### 类型安全（TypeScript）
- [ ] 类型定义是否完整？
- [ ] 是否避免使用any？
- [ ] 泛型使用是否合理？

### 性能
- [ ] 是否有性能问题？
- [ ] 是否有不必要的渲染？
- [ ] 内存泄漏风险？

### 可维护性
- [ ] 代码是否易读？
- [ ] 注释是否充分？
- [ ] 是否易于测试？

## Context
- 项目规范：[规范文档]
- 相关代码：[参考文件]
```
